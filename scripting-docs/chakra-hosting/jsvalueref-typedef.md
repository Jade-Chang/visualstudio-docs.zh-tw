# <a name="jsvalueref-typedef"></a>JsValueRef Typedef
JavaScript 值的參考。  
  
## <a name="syntax"></a>語法  
  
```  
typedef JsRef JsValueRef;  
```  
  
## <a name="remarks"></a>備註  
 JavaScript 值是下列其中一種類型的值：未定義、Null、布林值、字串、數值或物件。  
  
## <a name="requirements"></a>需求  
 **標頭：** jsrt.h
