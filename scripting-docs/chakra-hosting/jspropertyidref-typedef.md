# <a name="jspropertyidref-typedef"></a>JsPropertyIdRef Typedef
屬性識別項。  
  
## <a name="syntax"></a>語法  
  
```  
typedef JsRef JsPropertyIdRef;  
```  
  
## <a name="remarks"></a>備註  
 屬性識別項用於參考 JavaScript 物件的屬性，而非使用字串。  
  
## <a name="requirements"></a>需求  
 **標頭：** jsrt.h
