# <a name="iactivescriptprofilercallback3-interface"></a>IActiveScriptProfilerCallback3 介面
提供方法來通知分析工具，分析的執行的 web 背景工作的內容中發生。  
  
## <a name="syntax"></a>語法  
  
```  
interface IActiveScriptProfilerCallback3 : IActiveScriptProfilerCallback2  
```  
  
## <a name="methods"></a>方法  
  
|方法|說明|  
|------------|-----------------|  
|[IActiveScriptProfilerCallback3::SetWebWorkerId 方法](../../winscript/reference/iactivescriptprofilercallback3-setwebworkerid-method.md)|通知分析工具有關要用於此程式碼剖析工作階段的背景工作識別碼。|
