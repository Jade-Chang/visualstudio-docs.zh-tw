# <a name="iremotedebugapplicationthreadex-interface"></a>IRemoteDebugApplicationThreadEx 介面

 除了繼承自`IUnknown`、`IRemoteDebugApplicationThreadEx`介面會公開下列方法。  
  
## <a name="methods-in-vtable-order"></a>依照 Vtable 順序的方法  
  
|方法|說明|  
|------------|-----------------|  
|[IRemoteDebugApplicationThreadEx:EnumGlobalContexts](../../winscript/reference/iremotedebugapplicationthreadex-enumglobalcontexts.md)|列舉此執行緒中執行的所有語言通用的運算式內容。|
