 除了繼承自`IUnknown`、`IDebugAsyncOperationCallBack`介面會公開下列方法。  
  
|方法|說明|  
|------------|-----------------|  
|[IDebugAsyncOperationCallBack::onComplete](../../winscript/reference/idebugasyncoperationcallback-oncomplete.md)|發出訊號，結果是可從非同步的偵錯作業。|
