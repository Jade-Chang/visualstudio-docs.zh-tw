# <a name="idebugapplicationthreadevents110-onbeginthreadrequest"></a>IDebugApplicationThreadEvents110 ::OnBeginThreadRequest
呼叫到使用 PDM 執行緒的執行緒切換已開始。  
  
> [!IMPORTANT]
>  [IDebugApplicationThreadEvents110 介面](../../winscript/reference/idebugapplicationthreadevents110-interface.md)會實作由 PDM v11.0 和更新版本。 可在 activdbg100.h 中找到。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT OnBeginThreadRequest( void );  
```  
  
#### <a name="parameters"></a>參數  
 這個方法沒有任何參數。
