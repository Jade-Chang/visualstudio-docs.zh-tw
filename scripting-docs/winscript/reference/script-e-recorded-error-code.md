# <a name="scripterecorded-error-code"></a>SCRIPT_E_RECORDED 錯誤碼
已將指令碼引擎與主機之間傳遞錯誤。 主機必須傳遞給呼叫者的錯誤碼。  
  
## <a name="syntax"></a>語法  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
