# <a name="ijsdebugdatatargetreadbstr-method"></a>IJsDebugDataTarget::ReadBSTR 方法
讀取偵錯目標的 BSTR。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT ReadBSTR(  
   UINT64 address,  
   BSTR *pString  
);  
```  
  
#### <a name="parameters"></a>參數  
 `address`  
 [in]要讀取的位址。  
  
 `pString`  
 [out]BSTR 讀取偵錯目標。  
  
## <a name="return-value"></a>傳回值  
  
## <a name="remarks"></a>備註  
 如果位址不正確，請傳回 E_JsDEBUG_INVALID_MEMORY_ADDRESS。  
  
## <a name="requirements"></a>需求  
 **標頭：** jscript9diag.h
