# <a name="iactivescriptsiteinterruptpoll-interface"></a>IActiveScriptSiteInterruptPoll 介面
`IActiveScriptSiteInterruptPoll`介面可讓主應用程式指定指令碼應該結束。  
  
## <a name="methods"></a>方法  
 除了繼承自`IUnknown`、`IActiveScriptSiteInterruptPoll`介面會公開下列方法。  
  
|方法|說明|  
|------------|-----------------|  
|[IActiveScriptSiteInterruptPoll::QueryContinue](../../winscript/reference/iactivescriptsiteinterruptpoll-querycontinue.md)|可讓主應用程式指定指令碼應該結束。|
