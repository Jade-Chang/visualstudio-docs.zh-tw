# <a name="ienumjsstackframesnext-method"></a>IEnumJsStackFrames::Next 方法
取得指定的框架數。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT Next(  
   ULONG cFrameCount,  
   JS_NATIVE_FRAME *pFrames,  
   ULONG *pcFetched  
);  
```  
  
#### <a name="parameters"></a>參數  
 `cFrameCount`  
 [in]若要取得的畫面格數目。  
  
 `pFrames`  
 [out]要儲存框架的陣列。  
  
 `pcFetched`  
 [out]傳回的畫面格數目。  
  
## <a name="return-value"></a>傳回值

 **標頭：** jscript9diag.h
