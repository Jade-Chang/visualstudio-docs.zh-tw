# <a name="ijsenumdebugpropertynext-method"></a>IJsEnumDebugProperty::Next 方法
讀取此物件屬性。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT Next(  
   ULONG count,  
   IJsDebugProperty **ppDebugProperty,  
   ULONG *pActualCount  
);  
```  
  
#### <a name="parameters"></a>參數  
 `count`  
 [in]要讀取的屬性數目。  
  
 `ppDebugProperty`  
 [out]物件，表示屬性瀏覽器。  
  
 `pActualCount`  
 [out]物件的屬性的實際數目。  
  
## <a name="return-value"></a>傳回值  
  
## <a name="requirements"></a>需求  
 **標頭：** jscript9diag.h
