# <a name="ijsdebugbreakpointisenabled-method"></a>IJsDebugBreakPoint::IsEnabled 方法
判斷是否啟用中斷點。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT IsEnabled(  
   BOOL *pIsEnabled  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pIsEnabled`  
 [out]如果啟用中斷點，則傳回 true反之則傳回 false。  
  
## <a name="return-value"></a>傳回值  
  
## <a name="remarks"></a>備註  
 如果已刪除的中斷點上呼叫會傳回 E_UNEXPECTED。

 **標頭：** jscript9diag.h
