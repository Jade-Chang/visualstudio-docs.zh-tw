# <a name="ijsdebugpropertygetmembers-method"></a>IJsDebugProperty::GetMembers 方法
取得此物件的成員。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetMembers(  
   JS_PROPERTY_MEMBERS members,  
   IJsEnumDebugProperty **ppEnum  
);  
```  
  
#### <a name="parameters"></a>參數  
 `members`  
 [in]旗標來指定的成員資訊中包含的內容。  
  
 `ppEnum`  
 [out]物件的成員。  
  
## <a name="return-value"></a>傳回值  
  
## <a name="requirements"></a>需求  
 **標頭：** jscript9diag.h
