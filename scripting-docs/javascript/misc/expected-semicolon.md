# <a name="expected-3939"></a>必須是 &#39;&#39;
您嘗試將一個以上的陳述式放在一行中，但未不以分號隔開。 分號用來終止陳述式。 雖然您可以將多個陳述式放在單一行，每個必須從 [下一步] 以分號分隔。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   標記每個陳述式以分號的結尾。  
  
-   請確定您的函式呼叫正確使用括號。  
  
-   請確定您不忘記的標頭內分號**如**迴圈。
