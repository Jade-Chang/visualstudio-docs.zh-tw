# <a name="expected-3939-javascript"></a>必須是 &#39;) &#39;(JavaScript)
您嘗試將括在括號，一組運算式，但不是含右括號。 某些運算式必須括在一組左右括號內。 請注意括號，在下列範例中使用。  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   加入評估運算式的右括號。
