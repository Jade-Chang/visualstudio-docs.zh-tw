# <a name="object-doesn39t-support-this-property-or-method"></a>物件不&#39;支援此屬性或方法
您指定此物件沒有的屬性或方法。 並非所有物件都支援所有屬性和方法。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   檢查屬性和方法，確保沒有輸入錯誤。  
  
-   確認物件的執行階段類型是預期的物件類型。  
  
-   如需詳細資訊，請參閱物件的文件。
