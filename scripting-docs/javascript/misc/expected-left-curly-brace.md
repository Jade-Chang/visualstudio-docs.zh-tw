# <a name="expected-3939"></a>必須是 &#39;{&#39;
未包含標記的函式主體開頭的左大括號。 進行主體的函式，即使它是單一行中的程式碼必須包含左邊和右邊大括號內。 請注意，使用的函式主體中的大括號比在迴圈中更嚴格。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   加入標記的函式主體開頭的左大括號。
