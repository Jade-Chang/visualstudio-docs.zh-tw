# <a name="expected-3939"></a>必須是 &#39;: &#39;
您嘗試建立使用三元的條件運算子的運算式，但不是含第二個和第三個運算元之間的冒號。 三元 （三個運算元） 的條件運算子需要第二個 (true) 與第三個 (false) 之間冒號運算元。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   插入第二個和第三個運算元之間的冒號。
