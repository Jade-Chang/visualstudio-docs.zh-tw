# <a name="unterminated-string-constant-javascript"></a>未結束的字串常數 (JavaScript)
您並未結束字串常數，與為右雙引號。 字串常數必須括在引號內的一組內。  
  
> [!NOTE]
>  您可以使用成對的單引號或雙引號。 在以單引號括住的字串中可包含雙引號和單引號可以包含在以雙引號括住的字串。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   加入字串的結尾引號。
