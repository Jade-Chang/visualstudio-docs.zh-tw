# <a name="expected-39end39"></a>必須是 &#39;@end&#39;
您嘗試建立條件式編譯的區塊的程式碼，但不是含 **@end** 陳述式結尾。 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]陳述式可以有條件地編譯括住內 **@if/ @end** 區塊。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   新增對應 **@end** 陳述式。
