# <a name="expected-identifier-string-or-number"></a>必須是識別項、字串或數字
您用來宣告物件常值的常值的語法不正確。 物件常值的屬性必須是識別項、 字串或數字。 物件常值 （也稱為 「 物件初始設定 」） 是一份以逗號分隔的屬性： 值配對，所有括在方括號內。 例如：  
  
```JavaScript  
var point = {x:1.2, y:-3.4};  
```  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請使用適當的常值語法。
