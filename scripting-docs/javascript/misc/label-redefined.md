# <a name="label-redefined"></a>已重新定義標籤
您建立新的標籤，但會給它的現有標籤的名稱。 標籤可以用來標示程式碼區塊，但在指定範圍中，它們必須是唯一的。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   確定您在程式中使用的所有標籤它們個別的範圍內的唯一。
