# <a name="label-not-found"></a>找不到標籤
您所做的參考不存在的標籤。 標籤可用來標記程式碼，但指定的範圍內的區塊，必須是唯一的。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   檢查以確定未拼錯的標籤名稱。  
  
-   請確認標籤的所有參考不會都對目前的範圍 （這包括之前的定義） 中已定義的標籤。
