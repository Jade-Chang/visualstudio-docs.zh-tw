# <a name="expected-3939"></a>必須是 &#39;}&#39;
您未包含標示的函式主體、 迴圈、 程式碼，區塊的結尾的右大括號或物件初始設定式。 此錯誤的範例是**如**迴圈與只左大括號標記迴圈的主體。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   加入標記的結束函式、 迴圈、 區塊或物件初始設定式的右大括號。
