# <a name="boolean-expected"></a>必須是布林
您嘗試叫用**Boolean.prototype.toString**或**Boolean.prototype.valueOf**方法以外的類型的物件上`Boolean`。 這種類型的引動過程的物件必須屬於型別`Boolean`。 例如：  
  
```JavaScript  
var o = new Object;  
o.f = Boolean.prototype.toString;  
o.f();  
```
