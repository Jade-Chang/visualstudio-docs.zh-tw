# <a name="expected-3939-javascript"></a>必須是 &#39; = &#39;(JavaScript)
您嘗試建立用於條件式編譯陳述式、 變數，但未將變數與您想要為其指派的值之間的等號。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   加入等號。 例如:   
  
    ```JavaScript  
    @set @myvar1 = 1  
    ```
