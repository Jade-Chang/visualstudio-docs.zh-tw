# <a name="expected-3939"></a>必須是 &#39; @&#39;
您嘗試建立用於條件式編譯陳述式中使用變數`@set`陳述式，但未將 @ 記號"**@**"變數名稱之前。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   新增符號"**@**"之前的變數名稱。 例如:   
  
    ```JavaScript  
    @set @myvar = 1  
    ```
