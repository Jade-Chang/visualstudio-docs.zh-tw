# <a name="expected-3939"></a>必須是 &#39;]&#39;
您所做的參考陣列項目，但未包含右方括號。 陣列項目是指任何運算式必須包含同時左右方括號。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   加入參考陣列項目運算式的右括號。
