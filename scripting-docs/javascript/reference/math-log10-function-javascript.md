# <a name="mathlog10-function-javascript"></a>Math.log10 函式 (JavaScript)
傳回數字以 10 為底數的對數。  
  
## <a name="syntax"></a>語法  
  
```  
Math.log10(x)   
```  
  
## <a name="return-value"></a>傳回值  
 如果 `x` 是 `NaN`，此函式會傳回 `NaN`。 如果 `x` 小於零 (0)，此函式會傳回 `NaN`。
