# <a name="input-property--regexp-javascript"></a>input 屬性 ($_) (RegExp) (JavaScript)
傳回針對已執行規則運算式搜尋字串。 唯讀。  
  
## <a name="syntax"></a>語法  
  
```  
  
RegExp.input  
```  
  
## <a name="remarks"></a>備註  
 這個屬性與關聯的物件永遠是全域`RegExp`物件。  
  
 值**輸入**修改搜尋的字串是的變更的屬性。  
  
 下列範例說明使用**輸入**屬性：  
  
```JavaScript  
function inputDemo(){  
   var s;  
   var re = new RegExp("d(b+)(d)","ig");  
   var str = "cdbBdbsbdbdz";  
   var arr = re.exec(str);  
   s = "The string used for the match was " + RegExp.input;   
   return(s);  
}  
```
