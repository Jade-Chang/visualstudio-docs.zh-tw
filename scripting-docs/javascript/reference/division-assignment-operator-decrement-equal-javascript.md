# <a name="division-assignment-operator---javascript"></a>除法指派運算子 （-） (JavaScript)
將變數值除以運算式的值，然後將結果指派給此變數。  
  
## <a name="syntax"></a>語法  
  
```  
  
result /= expression   
```  
  
## <a name="parameters"></a>參數  
 `result`  
 任何數值變數。  
  
 `expression`  
 任何數值運算式。  
  
## <a name="remarks"></a>備註  
 使用`/=`運算子相當於指定： `result = result / expression`。
