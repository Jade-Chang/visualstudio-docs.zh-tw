# <a name="mathasinh-function-javascript"></a>Math.asinh 函式 (JavaScript)
傳回數字的反雙曲正弦。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.asinh(x)   
```  
  
## <a name="remarks"></a>備註  
 如果 `x` 是 `NaN`，結果為 `NaN`。  
  
 **適用於**: [Math 物件](../../javascript/reference/math-object-javascript.md)
