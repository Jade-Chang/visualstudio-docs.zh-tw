# <a name="mathclz32-function-javascript"></a>Math.clz32 函式 (JavaScript)
傳回數字之 32 位元二進位表示法中的前置零位元數。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.clz32(  
number  
)   
```  
  
## <a name="remarks"></a>備註  
 如果 `number` 是 0，則結果會是 32。 如果32 位元二進位編碼的最高有效位元 `number` 為 1，結果將會是 0。
