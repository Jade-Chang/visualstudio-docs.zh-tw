# <a name="assignment-operator--javascript"></a>指派運算子 (=) (JavaScript)
將值指派給變數。  
  
## <a name="syntax"></a>語法  
  
```  
  
result = expression  
```  
  
## <a name="parameters"></a>參數  
 `result`  
 任何變數。  
  
 `expression`  
 任何數值運算式。  
  
## <a name="remarks"></a>備註  
 = 運算子的行為方式如同其他運算子，讓它包含的運算式有一個值。 這表示您可以鏈結的指派運算子，如下所示： `j = k = l = 0`。 在此情況下`j`， `k`，和`l`等於零。
