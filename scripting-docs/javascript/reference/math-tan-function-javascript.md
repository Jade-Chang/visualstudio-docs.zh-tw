# <a name="mathtan-function-javascript"></a>Math.tan 函式 (JavaScript)
傳回一個數字的正切值。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## <a name="remarks"></a>備註  
 所需`radians`引數是包含以弧度角度的數值運算式。  
  
 傳回值是數值引數的正切函數`radians`。
