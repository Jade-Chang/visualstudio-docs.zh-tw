# <a name="void-operator-javascript"></a>void 運算子 (JavaScript)
避免運算式傳回值。  
  
## <a name="syntax"></a>語法  
  
```  
void expression   
```  
  
## <a name="remarks"></a>備註  
 `expression`引數是任何有效[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]運算式。  
  
 `void`運算子會評估運算式，並傳回`undefined`。 它是在評估運算式，但您不想將結果顯示指令碼的其餘部分的情況下很有用。
