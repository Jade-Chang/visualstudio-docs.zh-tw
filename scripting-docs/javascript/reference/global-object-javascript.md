# <a name="global-object-javascript"></a>全域物件 (JavaScript)
內建物件，其目的是為了收集到一個物件的全域函式和常數。  
  
## <a name="remarks"></a>備註  
 `Global`物件永遠不會直接使用，而且無法使用建立`new`運算子。 它會建立時初始化指令碼引擎時，因而建立它的函式和常數可立即使用。
