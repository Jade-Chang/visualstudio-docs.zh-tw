# <a name="mathsinh-function-javascript"></a>Math.sinh 函式 (JavaScript)
傳回數字的反雙曲正弦。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.sinh(x)   
```  
  
## <a name="remarks"></a>備註  
 傳回值與 `(exp(x) - exp(-x))/2` 相同。 如果 x 是 `NaN`，結果為 `NaN`。
