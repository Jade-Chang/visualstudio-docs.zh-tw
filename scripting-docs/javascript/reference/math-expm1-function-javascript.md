# <a name="mathexpm1-function-javascript"></a>Math.expm1 函式 (JavaScript)
傳回 e 減去 1 (自然對數的基數) 的乘冪。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.expm1(number)   
```  
  
## <a name="remarks"></a>備註  
 必要的 `number` 引數為數值運算式，代表 e 的乘冪。  
  
 傳回值為數字。 常數 e 為歐拉數，大約等於 2.71828，而 `number` 則為提供的引數。
