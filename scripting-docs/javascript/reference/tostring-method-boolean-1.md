# <a name="tostring-method-boolean-1"></a>toString 方法 （布林值） 1
傳回物件的字串表示。  
  
## <a name="syntax"></a>語法  
  
```  
  
boolean.toString()  
```  
  
## <a name="parameters"></a>參數  
 `boolean`  
 必要項。 要取得的字串表示的物件。  
  
## <a name="return-value"></a>傳回值  
 如果布林值為`true`，傳回"true"。 反之則傳回"false"。  
  
## <a name="remarks"></a>備註  
  
## <a name="example"></a>範例  
 下列範例說明使用**toString**方法。  
  
```JavaScript  
var s = new Boolean(0);  
document.write(s.toString());  
  
// Output: false;  
  
```
