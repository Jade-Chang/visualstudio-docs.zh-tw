# <a name="constructor-property-weakset"></a>constructor 屬性 (WeakSet)
指定用來建立 `WeakSet` 的函式。  
  
## <a name="syntax"></a>語法  
  
```JavaScript  
weakset.constructor  
```  
  
## <a name="remarks"></a>備註  
 必要的 `weakset` 是集合的名稱。  
  
 `constructor` 屬性是具有原型之每個物件的原型成員。 其中包括所有內建 JavaScript 物件，但不包括 `Global` 和 `Math` 物件。 `constructor` 屬性包含用來建構該特定物件執行個體的函式參考。
