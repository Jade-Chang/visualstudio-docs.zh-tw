# <a name="division-operator---javascript"></a>除法運算子 （-） (JavaScript)
會將兩個運算式的值。  
  
## <a name="syntax"></a>語法  
  
```  
  
result = number1 / number2   
```  
  
## <a name="parameters"></a>參數  
 `result`  
 任何數值變數。  
  
 `number1`  
 任何數值運算式。  
  
 `number2`  
 任何數值運算式。
