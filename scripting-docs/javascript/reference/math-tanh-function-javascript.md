# <a name="mathtanh-function-javascript"></a>Math.tanh 函式 (JavaScript)
傳回數字的雙曲線正切。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## <a name="remarks"></a>備註  
 傳回值與 `(exp(x) - exp(-x))/(exp(x) + exp(-x))` 相同。 如果 `x` 是 `NaN`，結果為 `NaN`。
