# <a name="size-property-map-javascript"></a>size 屬性 (Map) (JavaScript)
傳回對應中的項目數目。  
  
## <a name="syntax"></a>語法  
  
```JavaScript  
sizeVar = mapObj.size  
```  
  
#### <a name="parameters"></a>參數  
 `sizeVar`  
 必要項。 任何數字。  
  
 `mapObj`  
 必要項。 任何 `Map` 物件。
