# <a name="mathtrunc-function-javascript"></a>Math.trunc 函式 (JavaScript)
傳回數字的整數部分，並移除任何小數位數。  
  
## <a name="syntax"></a>語法  
  
```vb  
Math.trunc(number)  
```  
  
## <a name="remarks"></a>備註  
 必要的 `number` 引數是需要已截斷整數的數值運算式。  
  
 如果 `number` 是 `NaN`，則會傳回 `NaN`。
