# <a name="mathatan2-function-javascript"></a>Math.atan2 函式 (JavaScript)
傳回從 X 軸的角度 （以弧度為單位） 的點 (*y*，*x*)。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.atan2(y, x)  
```  
  
## <a name="parameters"></a>參數  
 *x*  
 必要項。 數值運算式，表示笛卡兒座標的 x 座標。  
  
 *y*  
 必要項。 數值運算式，表示笛卡兒座標的 y 座標。  
  
## <a name="remarks"></a>備註  
 傳回值是之間 *-pi*和*pi*。 它表示所提供的角度 (*y*，*x*) 點，以弧度為單位。
