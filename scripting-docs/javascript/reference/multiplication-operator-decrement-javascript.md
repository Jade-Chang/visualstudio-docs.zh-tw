# <a name="multiplication-operator--javascript"></a>乘法運算子 (*) (JavaScript)
乘上兩個運算式的值。  
  
## <a name="syntax"></a>語法  
  
```  
  
result = number1*number2  
```  
  
## <a name="parameters"></a>參數  
 *結果*  
 任何變數。  
  
 `number1`  
 任何運算式。  
  
 `number2`  
 任何運算式。
