# <a name="valueof-method-array"></a>valueOf 方法 (陣列)
傳回指定物件的基本值。  
  
## <a name="syntax"></a>語法  
  
```  
  
array.valueOf()  
```  
  
#### <a name="parameters"></a>參數  
 這個方法沒有任何參數。  
  
## <a name="return-value"></a>傳回值  
 傳回陣列的執行個體。  
  
## <a name="remarks"></a>備註  
 在下列範例中，具現化的陣列物件是這個方法的傳回值相同。  
  
```JavaScript  
var arr = [1, 2, 3, 4];  
var s = arr.valueOf();  
  
if (arr === s)  
document.write("same");  
else  
document.write("different");  
  
// Output:  
// same  
  
```
