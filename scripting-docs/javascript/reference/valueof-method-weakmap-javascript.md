# <a name="valueof-method-weakmap-javascript"></a>valueOf 方法 (WeakMap) (JavaScript)
傳回指定物件的基本值。  
  
## <a name="syntax"></a>語法  
  
```JavaScript  
weakmapObj.valueOf()  
```  
  
#### <a name="parameters"></a>參數  
 `weakmapObj`  
 必要項。 `WeakMap` 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 傳回`WeakMap`執行個體。
