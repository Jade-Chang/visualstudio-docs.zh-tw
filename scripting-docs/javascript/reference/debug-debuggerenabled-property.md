# <a name="debugdebuggerenabled-property"></a>Debug.debuggerEnabled 屬性
判斷是否啟用偵錯指令碼內容。 偵錯可能會啟用或停用，不論是否已附加偵錯工具。  
  
## <a name="syntax"></a>語法  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
