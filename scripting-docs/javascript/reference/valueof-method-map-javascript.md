# <a name="valueof-method-map-javascript"></a>valueOf 方法 (Map) (JavaScript)
傳回指定物件的基本值。  
  
## <a name="syntax"></a>語法  
  
```JavaScript  
mapObj.valueOf()  
```  
  
#### <a name="parameters"></a>參數  
 `mapObj`  
 必要項。 `Map` 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 傳回對應執行個體。
