# <a name="promiseresolve-function-promise"></a>Promise.resolve 函式 (承諾)
建立新的已解析承諾，其結果等同於其引數。  
  
## <a name="syntax"></a>語法  
  
```  
Promise.resolve(x)  
```  
  
#### <a name="parameters"></a>參數  
 `x`  
 必要項。 隨已完成的承諾傳回的值。  
  
## <a name="remarks"></a>備註  
 傳回已完成的 Promise 物件時，即會執行`then` 方法的履行處理函式。  
  
## <a name="example"></a>範例  
  
```JavaScript  
var p = Promise.resolve('success');  
p.then(function(result) {  
    console.log(result);  
});  
  
// Output:  
// success  
```
