# <a name="mathatan-function-javascript"></a>Math.atan 函式 (JavaScript)
傳回一個數字的反正切值。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.atan(number)   
```  
  
## <a name="remarks"></a>備註  
 所需`number`引數是數值運算式所需的反正切值。  
  
 傳回值是的反正切值`number`引數，以弧度為單位。
