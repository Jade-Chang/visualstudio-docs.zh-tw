# <a name="todatestring-method-date-javascript"></a>toDateString 方法 (日期) (JavaScript)
傳回字串值的日期。  
  
## <a name="syntax"></a>語法  
  
```  
  
objDate.toDateString( )  
```  
  
## <a name="remarks"></a>備註  
 所需*objDate*參考是`Date`物件。  
  
 `toDateString`方法會傳回字串值，包含目前的時區，在一個方便的日期，輕鬆地閱讀的格式。
