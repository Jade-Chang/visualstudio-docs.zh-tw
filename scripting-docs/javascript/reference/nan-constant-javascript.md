# <a name="nan-constant-javascript"></a>NaN 常數 (JavaScript)
特殊的常數，指定 thatNot 運算式不是數字。  
  
## <a name="syntax"></a>語法  
  
```  
NaN   
```  
  
## <a name="remarks"></a>備註  
 `NaN`常數 （不是數字） 是屬於`Global`物件，並初始化指令碼引擎時才可以提供。
