# <a name="valueof-method-set-javascript"></a>valueOf 方法 (Set) (JavaScript)
傳回指定物件的基本值。  
  
## <a name="syntax"></a>語法  
  
```JavaScript  
setObj.valueOf()  
```  
  
#### <a name="parameters"></a>參數  
 `setObj`  
 必要項。 `Set` 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 傳回集合執行個體。
