# <a name="mathceil-function-javascript"></a>Math.ceil 函式 (JavaScript)
大於或等於其數字的引數，則傳回的最小整數。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.ceil(number)   
```  
  
## <a name="remarks"></a>備註  
 所需要的 `number` 引數是數字運算式。  
  
 傳回值是整數值等於最小整數的值大於或等於其數字的引數。
