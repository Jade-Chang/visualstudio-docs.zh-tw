# <a name="mathlog1p-function-javascript"></a>Math.log1p 函式 (JavaScript)
傳回 1 + a 數字的自然對數。  
  
## <a name="syntax"></a>語法  
  
```  
Math.log1p(x)   
```  
  
## <a name="return-value"></a>傳回值  
 如果 `x` 是 `NaN`，此函式會傳回 `NaN`。 如果 `x` 小於零 (0)，此函式會傳回 `NaN`。
