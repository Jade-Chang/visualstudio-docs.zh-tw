# <a name="constructor-property-intlnumberformat"></a>constructor 屬性 (Intl.NumberFormat)
指定的函式，建立格式器。  
  
## <a name="syntax"></a>語法  
  
```JavaScript  
numberFormatter.constructor  
```  
  
## <a name="remarks"></a>備註  
 所需`numberFormatter`格式子的名稱。  
  
 `constructor` 屬性是具有原型之每個物件的原型成員。 其中包括所有內建 JavaScript 物件，但不包括 `Global` 和 `Math` 物件。 `constructor` 屬性包含用來建構該特定物件執行個體的函式參考。
