# <a name="mathcos-function-javascript"></a>Math.cos 函式 (JavaScript)
傳回一個數字的餘弦值。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.cos(radians)   
```  
  
## <a name="remarks"></a>備註  
 所需`radians`引數是包含以弧度角度的數值運算式。  
  
 傳回值是數字的引數的餘弦`radians`。
