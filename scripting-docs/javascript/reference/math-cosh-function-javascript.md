# <a name="mathcosh-function-javascript"></a>Math.cosh 函式 (JavaScript)
傳回數字的雙曲線餘弦。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.cosh(x)   
```  
  
## <a name="remarks"></a>備註  
 傳回值與 `(exp(x) + exp(-x))/2` 相同。 如果 `x` 是 `NaN`，結果為 `NaN`。
