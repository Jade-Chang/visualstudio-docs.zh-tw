# <a name="mathsin-function-javascript"></a>Math.sin 函式 (JavaScript)
傳回一個數字的正弦值。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.sin(radians)   
```  
  
## <a name="remarks"></a>備註  
 `radians`引數是包含以弧度角度的數值運算式。  
  
 傳回值是數字的引數的正弦值。
