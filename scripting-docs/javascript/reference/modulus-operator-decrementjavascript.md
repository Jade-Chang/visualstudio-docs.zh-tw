# <a name="modulus-operator--javascript"></a>模數運算子 (JavaScript)
將另一個數值運算式的值的數值運算式的值除以，並產生其餘部分。  
  
## <a name="syntax"></a>語法  
  
```  
result = expression1 % expression2  
```  
  
## <a name="arguments"></a>引數  
 `result`  
 任何變數。  
  
 `expression1`  
 任何數值運算式。  
  
 `expression2`  
 任何數值運算式。
