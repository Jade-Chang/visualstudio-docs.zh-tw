# <a name="size-property-set-javascript"></a>size 屬性 (Set) (JavaScript)
傳回集合中的項目數目。  
  
## <a name="syntax"></a>語法  
  
```JavaScript  
sizeVar = setObj.size  
```  
  
#### <a name="parameters"></a>參數  
 `sizeVar`  
 必要項。 任何數字。  
  
 `setObj`  
 必要項。 任何 `Set` 物件。
