# <a name="tostring-method-set-javascript"></a>toString 方法 (Set) (JavaScript)
傳回一組的字串表示。  
  
## <a name="syntax"></a>語法  
  
```JavaScript  
setObj.toString()  
```  
  
#### <a name="parameters"></a>參數  
 `setObj`  
 必要項。 `Set` 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 集合的字串表示。
