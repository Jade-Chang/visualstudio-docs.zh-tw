# <a name="mathsign-function-javascript"></a>Math.sign 函式 (JavaScript)
傳回數字的正負號，以指出數字為正數、負數或 0。  
  
## <a name="syntax"></a>語法  
  
```  
Math.sign(number)  
```  
  
## <a name="remarks"></a>備註  
 所需之 `number` 引數是須有正負號的數值運算式。  
  
 傳回值可以是下列其中之一：  
  
-   `NaN`，如果 `number` 是 `NaN`。  
  
-   -0，如果`number`是 0。  
  
-   +0，如果 `number` 是 +0。  
  
-   -1，如果`number`為負數，-0。  
  
-   +1，如果 `number` 是正值，而且不是 +0。
