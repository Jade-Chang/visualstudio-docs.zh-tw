# <a name="thread-ready-connector"></a>執行緒就緒連接器
當您按一下封鎖區段查看呼叫堆疊及其解除封鎖堆疊時，可能也會顯示執行緒就緒連接器。 如果解除封鎖事件發生在目前處理序中的另一個執行緒上，則執行緒就緒連接器會以視覺化方式識別讓封鎖的執行緒繼續執行的執行緒及執行區段。
