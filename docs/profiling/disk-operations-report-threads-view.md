# <a name="disk-operations-report-threads-view"></a>磁碟作業報告 (執行緒檢視)
磁碟作業報告顯示磁碟通道中的磁碟 I/O 作業。  
  
 代表在目前可見的時間範圍中正在進行程式碼剖析的處理序所發生每次磁碟存取，都會報告這項資訊︰  
  
-   執行磁碟存取之處理序的名稱和 PID  
  
-   存取磁碟的執行緒識別碼  
  
-   已存取的檔案名稱  
  
-   每個檔案的讀取次數  
  
-   讀取的位元組數目  
  
-   讀取延遲 (以毫秒為單位)  
  
-   寫入次數  
  
-   寫入的位元組數目  
  
-   寫入延遲 (以毫秒為單位)
