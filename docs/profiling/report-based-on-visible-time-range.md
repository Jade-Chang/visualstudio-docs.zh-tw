# <a name="report-based-on-visible-time-range"></a>以可見時間範圍為基礎的報表
[設定檔] 檢視顯示的報表是以目前可見的時間範圍和通道為基礎。 若要查看資料不同子集的詳細資訊，請按一下圖例中的項目。  
  
 您可以在這裡的資料表中找到資料的詳細資訊。
