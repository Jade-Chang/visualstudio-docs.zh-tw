# <a name="idiaenumdebugstreamdatareset"></a>IDiaEnumDebugStreamData::Reset
重設為列舉的序列的開頭。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>傳回值  
 傳回 `S_OK`。
