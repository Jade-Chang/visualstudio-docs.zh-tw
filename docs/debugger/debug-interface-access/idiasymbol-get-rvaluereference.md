# <a name="idiasymbolgetrvaluereference"></a>IDiaSymbol::get_RValueReference
擷取指定為指標類型是否為右值參考的旗標。 使用時機[SymTagEnum 列舉](../../debugger/debug-interface-access/symtagenum.md)設為指標類型。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_RValueReference (  
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回`TRUE`如果指標為右值參考; 否則傳回`FALSE`。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。  
  
> [!NOTE]
>  傳回值為`S_FALSE`表示屬性不適用於符號。

 標頭： Dia2.h  
  
 程式庫： diaguids.lib  
  
 DLL: msdia100.dll
