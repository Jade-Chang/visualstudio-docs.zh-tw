# <a name="idiaenuminjectedsourcesskip"></a>IDiaEnumInjectedSources::Skip
略過指定的數目的列舉順序中的插入來源。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT Skip (   
   ULONG celt  
);  
```  
  
#### <a name="parameters"></a>參數  
 celt  
 [in]略過列舉順序中的插入來源數目。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回`S_FALSE`如果沒有其他插入的來源，以略過。
