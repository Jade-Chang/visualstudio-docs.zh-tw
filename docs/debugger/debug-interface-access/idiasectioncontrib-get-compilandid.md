# <a name="idiasectioncontribgetcompilandid"></a>IDiaSectionContrib::get_compilandId
擷取 > 一節的編譯模組識別項。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_compilandId (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回一節的編譯模組識別項。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`。 傳回`S_FALSE`不支援這個屬性，則為。 反之則傳回錯誤碼。
