# <a name="idiastackframegetbase"></a>IDiaStackFrame::get_base
擷取框架的基底位址。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_base (   
   ULONGLONG* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回基底地址。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`。 傳回`S_FALSE`不支援的屬性。 反之則傳回錯誤碼。
