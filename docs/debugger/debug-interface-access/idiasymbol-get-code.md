# <a name="idiasymbolgetcode"></a>IDiaSymbol::get_code
擷取符號是否參考程式碼位址所指定的旗標。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_code (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回`TRUE`如果符號參考程式碼位址，否則會傳回`FALSE`。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。  
  
> [!NOTE]
>  傳回值為`S_FALSE`表示屬性不適用於符號。
