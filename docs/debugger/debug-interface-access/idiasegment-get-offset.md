# <a name="idiasegmentgetoffset"></a>IDiaSegment::get_offset
擷取區段，區段的開始處的位移。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_offset (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回位移，區段，區段開始的位置。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`。 傳回`S_FALSE`不支援這個屬性，則為。 反之則傳回錯誤碼。
