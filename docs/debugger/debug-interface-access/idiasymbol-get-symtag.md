# <a name="idiasymbolgetsymtag"></a>IDiaSymbol::get_symTag
擷取符號類型分類。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_symTag (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回值，從[SymTagEnum 列舉](../../debugger/debug-interface-access/symtagenum.md)列舉，指定符號類型分類。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。  
  
> [!NOTE]
>  傳回值為`S_FALSE`表示屬性不適用於符號。  
  
## <a name="example"></a>範例  
  
```C++  
IDiaSymbol* pType;  
DWORD       tag = 0;  
pType->get_symTag( &tag );  
```
