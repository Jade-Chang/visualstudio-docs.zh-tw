# <a name="idiasegmentgetread"></a>IDiaSegment::get_read
擷取指出是否可以讀取區段的旗標。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_read (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回`TRUE`如果可以讀取區段; 否則傳回`FALSE`。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`。 傳回`S_FALSE`不支援這個屬性，則為。 反之則傳回錯誤碼。
