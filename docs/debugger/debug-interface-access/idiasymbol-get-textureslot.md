# <a name="idiasymbolgettextureslot"></a>IDiaSymbol::get_textureSlot
擷取紋理位置。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_textureSlot(   
   DWORD* pRetVal);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]指標`DWORD`保存紋理位置。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。
