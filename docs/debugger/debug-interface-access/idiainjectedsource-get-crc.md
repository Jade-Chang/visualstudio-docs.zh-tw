# <a name="idiainjectedsourcegetcrc"></a>IDiaInjectedSource::get_crc
擷取循環冗餘檢查 (CRC) 計算從來源程式碼的位元組。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_crc (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回 CRC 計算從來源程式碼的位元組。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`。 傳回`S_FALSE`不支援這個屬性，則為。 反之則傳回錯誤碼。
