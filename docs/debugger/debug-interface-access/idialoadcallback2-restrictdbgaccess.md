# <a name="idialoadcallback2restrictdbgaccess"></a>IDiaLoadCallback2::RestrictDBGAccess
決定尋找偵錯資訊是否允許從.dbg 檔案。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT RestrictDBGAccess();  
```  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回錯誤碼。  
  
## <a name="remarks"></a>備註  
 任何傳回值非`S_OK`防止尋找.dbg 檔案從偵錯資訊。
