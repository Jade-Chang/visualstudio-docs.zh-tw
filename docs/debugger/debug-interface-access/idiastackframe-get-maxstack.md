# <a name="idiastackframegetmaxstack"></a>IDiaStackFrame::get_maxStack
擷取推入堆疊框架中的位元組的數目上限。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_maxStack (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回推送到堆疊上的位元組的數目上限。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`。 傳回`S_FALSE`不支援的屬性。 反之則傳回錯誤碼。
