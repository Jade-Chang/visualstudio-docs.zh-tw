# <a name="idiasymbolgetbuiltinkind"></a>IDiaSymbol::get_builtInKind
擷取的 HLSL 類型內建類型。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_buildInKind(   
   DWORD* pRetVal);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]指標`DWORD`保存內建的 HLSL 型別。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。
