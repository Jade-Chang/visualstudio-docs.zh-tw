# <a name="idiaenumlinenumbersskip"></a>IDiaEnumLineNumbers::Skip
略過指定的數目的列舉順序中的行號。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT Skip (   
   ULONG celt  
);  
```  
  
#### <a name="parameters"></a>參數  
 celt  
 [in]將列舉序列中的行號，以略過數目。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回`S_FALSE`如果沒有更多的行號，以略過。
