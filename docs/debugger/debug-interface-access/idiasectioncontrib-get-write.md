# <a name="idiasectioncontribgetwrite"></a>IDiaSectionContrib::get_write
擷取的旗標，指出是否可以修改一節。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_write (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回`TRUE`如果區段可寫入，否則傳回`FALSE`。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`。 傳回`S_FALSE`不支援這個屬性，則為。 反之則傳回錯誤碼。
