# <a name="idiaenumframedataskip"></a>IDiaEnumFrameData::Skip
略過指定的數目的框架列舉順序中的資料元素。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT Skip (   
   ULONG celt  
);  
```  
  
#### <a name="parameters"></a>參數  
 celt  
 [in]略過列舉順序中的框架資料元素的數目。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回`S_FALSE`如果沒有更多略過記錄。
