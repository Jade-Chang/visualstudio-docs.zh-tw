# <a name="idiasectioncontribgetdatacrc"></a>IDiaSectionContrib::get_dataCrc
擷取資料的區段中的循環冗餘檢查 (CRC)。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_dataCrc (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回資料的 CRC 一節。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`。 傳回`S_FALSE`不支援這個屬性，則為。 反之則傳回錯誤碼。
