# <a name="vsgdbgvsgdbg-destructor"></a>VsgDbg::~VsgDbg (解構函式)
終結的執行個體`VsgDbg`類別。 如果目前正在錄製圖形資訊，圖形記錄檔已完成並關閉，且主動擷取圖形資訊時所使用的資源會釋放。  
  
## <a name="syntax"></a>語法  
  
```C++  
~VsgDbg();  
```
