# <a name="select-code-type-dialog-box"></a>選取程式碼類型對話方塊
若要開啟此對話方塊中，開啟**附加至處理序**對話方塊，然後按一下**選取** 按鈕。  
  
 **自動判斷要偵錯程式碼的類型**  
 將依據正在執行的程式碼類型，選取適當的偵錯工具。  
  
 **偵錯這些程式碼類型：**  
 從提供的清單選擇要進行偵錯的程式碼類型。
