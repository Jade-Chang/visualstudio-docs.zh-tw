# <a name="process-search-dialog-box"></a>處理序搜尋對話方塊
使用此對話方塊，即可尋找並選取特定的處理序中的節點[處理序檢視](../debugger/processes-view.md)。 若要顯示此對話方塊中，焦點移至**處理程序**檢視 視窗中。 然後選擇 **尋找處理序**從**搜尋**功能表。  
  
 可用的設定如下：  
  
 **Process**  
 要搜尋的處理序識別碼。  
  
 **模組**  
 要搜尋的模組字串。  
  
 **向上或向下搜尋方向**  
 搜尋開始的方向。
