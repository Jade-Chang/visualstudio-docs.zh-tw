# <a name="message-search-dialog-box"></a>訊息搜尋對話方塊
使用此對話方塊，即可尋找並選取的節點中的特定訊息[訊息檢視](../debugger/messages-view.md)。 若要顯示此對話方塊中，將焦點移至 [訊息] 檢視視窗。 然後選擇 **尋找訊息**從**搜尋**功能表。  
  
 可用的設定如下：  
  
 **搜尋工具**  
 用來選取視窗拖放工具。  
  
 **隱藏 Spy + +**  
 移除主要 Spy + + 視窗，並將保留**MessageSearch**對話方塊在其他應用程式的上層。  
  
 **控制代碼**  
 要搜尋之訊息的唯一識別碼。  
  
 **Type**  
 要搜尋的訊息類型。  
  
 **訊息**  
 要搜尋的訊息識別碼。  
  
 **向上或向下搜尋方向**  
 搜尋開始的方向。
