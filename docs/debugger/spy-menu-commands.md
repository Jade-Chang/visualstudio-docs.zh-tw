# <a name="spy-menu-commands"></a>Spy 功能表命令
**Spy**功能表提供使用 Spy + + 視窗和檢視的命令。  
  
 **Windows**  
 開啟新視窗檢視。  
  
 **處理序**  
 開啟新的處理序檢視。  
  
 **執行緒**  
 開啟新執行緒檢視。  
  
 **記錄檔訊息**  
 開啟新的訊息檢視，並開啟[訊息選項對話方塊](../debugger/message-options-dialog-box.md)檢視。  
  
 **尋找視窗**  
 顯示**FindWindow**對話方塊來選取視窗中的，使用滑鼠。  
  
 **結束**  
 關閉 Spy + + 公用程式。
