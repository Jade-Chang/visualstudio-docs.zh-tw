# <a name="thread-search-dialog-box"></a>執行緒搜尋對話方塊
使用此對話方塊，即可尋找並選取特定的執行緒中的節點[執行緒檢視](../debugger/threads-view.md)。 若要顯示此對話方塊中，焦點移至**執行緒**檢視 視窗中。 然後選擇 **尋找執行緒**從**搜尋**功能表。  
  
 可用的設定如下：  
  
 **Thread**  
 要搜尋的執行緒識別碼。  
  
 **模組**  
 要搜尋的模組字串。  
  
 **向上或向下搜尋方向**  
 搜尋開始的方向。
