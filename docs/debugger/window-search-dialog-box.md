# <a name="window-search-dialog-box"></a>視窗搜尋對話方塊
使用此對話方塊，即可尋找並選取特定的視窗中的節點[視窗檢視](../debugger/windows-view.md)。 若要顯示此對話方塊中，將焦點移至 Windows 的 [檢視] 視窗。 然後選擇 **尋找視窗**從**搜尋**功能表。  
  
 您可以使用**搜尋工具**選取視窗中，或者您可以輸入視窗控制代碼，或識別的視窗的標題和類別。  
  
 可用的設定如下：  
  
 **搜尋工具**  
 用來選取視窗拖放工具。  
  
 **隱藏 Spy + +**  
 移除主要 Spy + + 視窗，並將保留**WindowSearch**對話方塊在其他應用程式的上層。  
  
 **控制代碼**  
 搜尋視窗的唯一識別碼。  
  
 **標題**  
 中要搜尋的視窗標題的文字。  
  
 **類別**  
 要搜尋的視窗類別。  
  
 **向上或向下搜尋方向**  
 搜尋開始的方向。
