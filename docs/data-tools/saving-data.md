# <a name="saving-data"></a>儲存資料
將資料儲存為保存的程序變更回原始的資料存放區，通常是例如 SQL Server 關聯式資料庫的應用程式的資料模型中的資料。  
  
 資料模型透過更新資料來源通常是兩步驟程序。 第一個步驟是以新資訊更新的資料模型，新的記錄，變更記錄，或刪除記錄。 第二個步驟是將變更儲存回資料庫的資料模型中。
