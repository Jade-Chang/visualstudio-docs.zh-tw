# <a name="clickonce-deployment-samples-and-walkthroughs"></a>ClickOnce 部署範例和逐步解說
本節包含範例應用程式、 程式碼範例和逐步解說，說明語法、 結構和用來部署 Windows Forms、 WPF 和主控台應用程式的技術。  
  
 範例程式碼僅供說明用途，並不能用於部署的方案，而不需修改。 特別是，安全性必須納入大於考量。
