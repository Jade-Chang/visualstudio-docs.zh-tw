# <a name="modules"></a>模組
偵錯工具就架構而言，**模組**:  
  
-   是程式碼，例如可執行檔或 DLL 的實體容器。  
  
-   可以重新載入其符號，並描述自己。 模組描述會顯示在 IDE 的 [模組] 視窗中。  
  
-   由[IDebugModule2](../../extensibility/debugger/reference/idebugmodule2.md)介面，以描述模組的偵錯引擎所建立。
