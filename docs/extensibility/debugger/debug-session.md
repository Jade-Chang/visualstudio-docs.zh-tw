# <a name="debug-session"></a>偵錯工作階段
偵錯工具就架構而言，**工作階段**:  
  
-   為所有的設定或處理程序，相關不相關，已啟動或附加至從 IDE 的單一執行個體。  
  
-   是由工作階段偵錯管理員 (SDM) 實作。 如需詳細資訊，請參閱[工作階段偵錯管理員](../../extensibility/debugger/session-debug-manager.md)。
