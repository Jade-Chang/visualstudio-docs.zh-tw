# <a name="idebugprogramprovider2setlocale"></a>IDebugProgramProvider2::SetLocale
建立要用於任何地區設定特定資源的地區設定。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SetLocale(  
   WORD wLangID  
);  
```  
  
```csharp  
int SetLocale(  
   ushort wLangID  
);  
```  
  
#### <a name="parameters"></a>參數  
 `wLangID`  
 [in]若要建立語言識別碼。 例如，1033 代表英文。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回錯誤碼。
