# <a name="idebugprocess2detach"></a>IDebugProcess2::Detach
會從這個處理序偵錯工具中斷連結卸離所有處理序中的程式。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Detach(   
   void   
);  
```  
  
```csharp  
int Detach();  
```  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回錯誤碼。  
  
## <a name="remarks"></a>備註  
 所有程式和處理程序繼續執行，但不會再偵錯工作階段的一部分。 在卸離後作業已完成，沒有其他偵錯此處理程序 （以及它的程式） 的事件將會傳送。
