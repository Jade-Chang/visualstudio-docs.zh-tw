# <a name="bptype"></a>BP_TYPE
指定中斷點是在程式碼的位置、 資料位置，還是其他類型的中斷點。  
  
## <a name="syntax"></a>語法  
  
```cpp  
enum enum_BP_TYPE {   
   BPT_NONE    = 0x0000,  
   BPT_CODE    = 0x0001,  
   BPT_DATA    = 0x0002,  
   BPT_SPECIAL = 0x0003  
};  
typedef DWORD BP_TYPE;  
```  
  
```csharp  
public enum enum_BP_TYPE {   
   BPT_NONE    = 0x0000,  
   BPT_CODE    = 0x0001,  
   BPT_DATA    = 0x0002,  
   BPT_SPECIAL = 0x0003  
};  
```  
  
## <a name="members"></a>成員  
 BPT_NONE  
 不指定任何中斷點類型。  
  
 BPT_CODE  
 指定程式碼中斷點。  
  
 BPT_DATA  
 指定資料中斷點。  
  
 BPT_SPECIAL  
 指定中斷點不是程式碼和資料類型。 此類型已被取代，不應使用。
