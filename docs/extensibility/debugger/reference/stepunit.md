# <a name="stepunit"></a>STEPUNIT
逐步執行步驟單位指定。  
  
## <a name="syntax"></a>語法  
  
```cpp  
enum enum_STEPUNIT {   
   STEP_STATEMENT   = 0,  
   STEP_LINE        = 1,  
   STEP_INSTRUCTION = 2  
};  
typedef DWORD STEPUNIT;  
```  
  
```csharp  
enum enum_STEPUNIT {   
   STEP_STATEMENT   = 0,  
   STEP_LINE        = 1,  
   STEP_INSTRUCTION = 2  
};  
```  
  
## <a name="members"></a>成員  
 STEP_STATEMENT  
 由陳述式的步驟。  
  
 STEP_LINE  
 所列的步驟。  
  
 STEP_INSTRUCTION  
 依指示的步驟。  
  
## <a name="remarks"></a>備註  
 做為引數傳遞[步驟](../../../extensibility/debugger/reference/idebugprocess3-step.md)方法。
