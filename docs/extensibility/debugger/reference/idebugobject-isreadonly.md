# <a name="idebugobjectisreadonly"></a>IDebugObject::IsReadOnly
判斷此物件是否為唯讀。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT IsReadOnly(   
   BOOL* pfIsReadOnly  
);  
```  
  
```csharp  
int IsReadOnly(  
   out int pfIsReadOnly  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pfIsReadOnly`  
 [out]傳回非零 (`TRUE`) 如果此物件是唯讀的; 否則傳回零 (`FALSE`)。  
  
## <a name="return-value"></a>傳回值  
 如果成功，會傳回 S_OK;反之則傳回錯誤碼。  
  
## <a name="remarks"></a>備註  
 唯讀的物件不能有它建立之後變更其值。
