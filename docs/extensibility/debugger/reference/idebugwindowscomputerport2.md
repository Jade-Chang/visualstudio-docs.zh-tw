# <a name="idebugwindowscomputerport2"></a>IDebugWindowsComputerPort2
可讓目標電腦的相關資訊的查詢。  
  
## <a name="syntax"></a>語法  
  
```  
IDebugWindowsComputerPort2 : IUnknown  
```

 實作這個介面是由偵錯工作階段管理員的連接埠物件。
