# <a name="add-commands-to-toolbars"></a>將命令加入至工具列
工具列是水平或垂直的區域，其中包含繫結至命令的按鈕。 您可以將您自己的命令加入 Visual Studio 中許多不同的 windows 上的工具列： Visual Studio 主視窗、 [方案總管] 中和您自己的自訂工具視窗。  
  
 本節說明一些您可以使用工具列上的命令執行的動作。
