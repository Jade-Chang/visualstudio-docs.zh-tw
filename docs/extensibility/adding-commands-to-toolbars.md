# <a name="adding-commands-to-toolbars"></a>將命令加入至工具列
工具列是包含繫結至命令按鈕的水平或垂直區域。 您可以將您自己的命令加入 Visual Studio 中許多不同的 windows 上的工具列： Visual Studio 主視窗、 [方案總管] 中和您自己的自訂工具視窗。  
  
 本節說明一些您可以使用工具列上的命令執行的狀況。
