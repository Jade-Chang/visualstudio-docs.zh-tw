# <a name="c6274"></a>C6274
警告 C6274： 當做參數傳遞給非字元\<數字 > 呼叫中需要字元\<函式 >

 這則警告表示的格式字串指定所需的字元 (比方說，`%c`或`%C`規格)，但是會傳遞非整數例如浮點數、 字串或結構。 這項缺失很可能會導致不正確的輸出。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str);
}
```

 若要修正這個警告，請使用下列程式碼：

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str[0]);
}
```

 下列程式碼會使用安全字串操作函數， `sprintf_s`，若要更正這個警告：

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf_s(buff,5,"%c", str[0]);
}
```
