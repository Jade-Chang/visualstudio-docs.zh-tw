# <a name="c28235"></a>C28235
警告 C28235： 函式不能獨立成為註釋

 這個警告表示嘗試使用未宣告為註解的註解的內容中的函式。 這包括獨立內容中使用的基本作業 (PrimOp)。 這不能進行註解使用標準巨集。 這個警告不會出現在一般使用。
