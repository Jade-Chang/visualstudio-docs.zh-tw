# <a name="c28273"></a>C28273
警告 C28273： 函式，線索與不一致的函式宣告

 這個警告表示發生錯誤，而非在正在分析的程式碼中的註解。 出現在函式定義上的註解是與出現在宣告不一致。 要比對必須解決的兩個註解。
