# <a name="c28221"></a>C28221
警告 C28221： 註解中的參數需要字串運算式  
  
 這個警告表示為註解參數必須是字串，且發現某些其他類型。 這通常表示編碼錯誤的註解巨集，並不會出現在正常使用。
