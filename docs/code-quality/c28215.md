# <a name="c28215"></a>C28215
警告 C28215: typefix 不能套用到已經有 typefix 的參數

 套用`__typefix`已經有該註釋的參數註釋是錯誤。 `__typefix`註解僅能用於幾個特殊情況下，而且這個警告不應該出現在正常使用。
