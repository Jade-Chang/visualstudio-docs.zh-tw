# <a name="ca0502"></a>CA0502
不支援的語言

 當使用者嘗試分析 ASP.NET 專案包含一個或多個頁面或以 C# 或 Visual Basic 以外的語言撰寫的類別時，就會發生此錯誤。 若要修正這個錯誤，將檔案從專案移除或停用程式碼分析。
