# <a name="c28024"></a>C28024
警告 C28024： 函式指標指派至已標註函式類別\<類別 >，其不包含在函式類別\</classlist >。  
  
 這兩個函數已標註函式類別，但不是符合類別時，會報告這個警告。  
  
 函式指標時，會發出這個警告`_Function_class_`指定只有特定的功能類別的函式，應該指派給它的註解。 在指派或隱含的指派函式呼叫中，來源和目標必須是相同的函式類別，但不是相符的函式類別。
