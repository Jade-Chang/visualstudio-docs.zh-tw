# <a name="c28304"></a>C28304
警告 C28304： 為 c + + 參考參數 < parameter_name >、 不當放置`_Notref_`運算子找不到套用至\<權杖 >。

 `_Notref_`運算子應該只用在涉及 c + + 參考參數的特殊情況下，只有在系統提供的巨集。 它必須緊接著`_Deref_`運算子或功能的註解。
