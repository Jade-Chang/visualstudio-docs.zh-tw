# <a name="ca0504"></a>CA0504
已忽略規則目錄，因為它不存在  
  
 如果在指定的目錄，就會產生警告 CA504 **CodeAnalysisRuleDirectories**找不到專案或建置的組態檔中的屬性。 若要解決此問題，請輸入有效的目錄路徑。
