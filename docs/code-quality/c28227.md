# <a name="c28227"></a>C28227
警告 C28227: 註釋不能也是 PrimOp (參閱之前的宣告)

 這個警告表示具名註解所宣告做為一般的附註，並也先前宣告為 PrimOp。 這不能進行註解; 使用標準巨集這個警告不會出現在一般使用。
