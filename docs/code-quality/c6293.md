# <a name="c6293"></a>C6293
警告 C6293: for 迴圈定義錯誤： 從最小值向下計數  
  
 這個警告表示預期的迴圈可能無法運作。 迴圈從最少，向下計數，但具有較高的終止條件時，它就會發生。  
  
 帶正負號，或不帶正負號，以及負值的遞增的索引變數將會造成迴圈的負值計數，直到發生溢位。 這將會結束迴圈。  
  
## <a name="example"></a>範例  
 下列範例程式碼會產生這個警告：  
  
```  
void f( )  
{  
   signed char i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
  
```  
  
 若要修正這個警告，請使用下列程式碼：  
  
```  
void f( )  
{  
   signed char i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
