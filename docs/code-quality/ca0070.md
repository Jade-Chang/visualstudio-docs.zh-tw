# <a name="ca0070"></a>CA0070
當程式碼分析分析組件，並遇到權限屬性而且至少一個下列條件為 true，則會發生這個錯誤：  
  
-   程式碼分析找不到包含屬性的組件。  
  
-   程式碼分析並沒有載入組件的權限。  
  
-   載入程式碼分析的組件不包含屬性。  
  
 在大部分情況下，您可以藉由確定您有最新版本的程式碼分析工具，解決的問題。
