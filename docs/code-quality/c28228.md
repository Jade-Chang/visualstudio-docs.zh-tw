# <a name="c28228"></a>C28228
警告 C28228： 註釋參數： 不能使用類型註釋中  
  
 這個警告表示參數是不支援的型別。 註解做為參數只能使用一組有限的類型。 這不能進行註解; 使用標準巨集這個警告不會出現在一般使用。
