# <a name="ca0505"></a>CA0505
已被取代的屬性會被忽略，因為 superceding 屬性定義。  
  
 同時指定已被取代而且包含在專案中的值取代程式碼分析屬性，或建置組態檔，就會產生警告 CA505。 若要解決此問題，請移除已被取代的屬性。
