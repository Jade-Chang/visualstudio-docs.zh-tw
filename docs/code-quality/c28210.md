# <a name="c28210"></a>C28210
警告 28210： 註釋`_On_failure_`內容不能在明確 pre context 中  
  
 註解`_On_failure_`必須明確或隱含指定在`__post`內容，也就是函式傳回後套用。  使用`_drv_out`確保這。
