# <a name="c6385"></a>C6385
警告 C6385： 無效的資料： 存取\<緩衝區名稱 >，可讀取的大小是\<size1 > 個位元組，但\<size2 > 可能讀取的位元組： 行： x、 y  
  
 這個警告表示指定之緩衝區的可讀取的範圍可能會小於用來從佇列讀取的索引。 嘗試讀取超出有效範圍的資料會導致緩衝區滿溢。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
void f(int i)  
{  
   char a[20];  
   char j;  
   if (i <= 20)  
   {  
      j = a[i];  
   }  
}  
```  
  
 若要修正這個警告，請使用下列程式碼：  
  
```  
void f(int i)  
{  
   char a[20];  
   char j;  
   if (i < 20)  
   {  
      j = a[i];  
   }  
}   
```
