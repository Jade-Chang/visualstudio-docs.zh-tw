# <a name="c28351"></a>C28351
警告 C28351： 註解\<註解 > 描述條件中無法使用動態值 （變數）。

 正在檢查 C/c + + 類型套用註釋時，會報告這個警告。

 中的運算式`_When_`應該評估為常數。 `_When_`會被忽略。
