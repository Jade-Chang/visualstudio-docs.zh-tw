# <a name="c28280"></a>C28280
警告 C28280:，'end' 找到符號，但沒有對應的 'begin'  
  
 註釋語言支援 begin 和 end (`{`和`}`c) 建構，並發現不對稱。 您可以避免這種情況下，如果使用巨集。
