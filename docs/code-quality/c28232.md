# <a name="c28232"></a>C28232
警告 C28232: _Pre\_，_Post\_，或 _Deref\_不會套用至任何註釋

 這個警告表示`_Pre_`， `_Post_`，或`_Deref_`運算子會出現在後續的功能性註釋沒有註釋運算式; 已忽略修飾詞，但這表示不正確地自動程式碼註解。
