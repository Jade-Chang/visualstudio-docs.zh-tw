# <a name="c28225"></a>C28225
警告 C28225： 註釋中找不到必要的參數的正確數目  
  
 這個警告表示具名註釋搭配參數數目不正確。 這不能進行註解; 使用標準巨集這個警告不會出現在一般使用。
