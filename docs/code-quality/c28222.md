# <a name="c28222"></a>C28222
警告 28222: （_y)\_，（_n)\_，或 _Maybe\_註釋需要

 這則警告表示為註解參數必須是其中一個符號是`_Yes_`， `_No_`，或`_Maybe_`，並遇到一些其他符號。 這通常表示編碼錯誤的註解巨集。
