# <a name="ca0057"></a>CA0057
不能對應至目前資料集的目標和規則的違規

 如果 FxCopCmd.exe 用來叫用手動的專案檔，就會發生這個錯誤或 / 匯入 XML 報告中具有無法對應至目前資料集的目標和規則的違規。

 在進行分析時，才會發生使用命令列工具的版本。
