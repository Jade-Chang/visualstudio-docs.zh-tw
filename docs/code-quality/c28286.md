# <a name="c28286"></a>C28286
警告 C28286： 函式，結尾附近發生語法錯誤

 程式碼分析工具會報告這個警告時可能發生錯誤時模型檔案中。 幾個來源檔案錯誤也可能導致此類錯誤。
