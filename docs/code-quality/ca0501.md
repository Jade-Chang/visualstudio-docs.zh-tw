# <a name="ca0501"></a>CA0501
無法讀取輸出報告  
  
 會發生這種情況時無法寫入輸出報表，因為下列原因之一：  
  
-   磁碟已滿  
  
-   沒有可寫入目錄的權限  
  
-   目錄不存在
