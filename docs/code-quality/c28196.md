# <a name="c28196"></a>C28196
警告 C28196： 無法滿足的需求。 （運算式不會評估為 true。）

 這則警告表示所分析函式之函式原型的 `__notnull` 參數或傳回值有 `__null`、`__drv_valueIs` 或 `_Out_`，不過傳回的值與該註釋不一致。
