# <a name="c28218"></a>C28218
警告 C28218： 函式參數，註釋的參數不符合找到的檔案  
  
 目前這一行，並在訊息中的列上的註釋不一致。 這不能進行註解; 使用標準巨集這個警告不會出現在正常使用。
