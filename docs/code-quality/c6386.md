# <a name="c6386"></a>C6386
警告 C6386： 緩衝區滿溢： 存取\<緩衝區名稱 >，可寫入的大小是\<size1 > 個位元組，但\<size2 > 可能會被寫入的位元組： 線條： x，y  
  
 這個警告表示指定之緩衝區的可寫入的範圍可能會小於用來寫入其中的索引。 這可能會造成緩衝區溢位。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這項警告和[C6201](../code-quality/c6201.md):  
  
```  
#define MAX 25  
  
void f ( )  
{  
  char ar[MAX];  
  //Code ...  
  ar[MAX] = '\0';  
}  
```  
  
 若要更正這兩個警告，請使用下列程式碼：  
  
```  
#define MAX 25  
  
void f ( )  
{  
   char a[MAX];  
   // code...  
   a[MAX - 1] = '\0';  
}   
```
