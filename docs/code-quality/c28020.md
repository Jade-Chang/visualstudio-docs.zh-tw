# <a name="c28020"></a>C28020
警告 C28020： 運算式\<e x p > 不是在此呼叫，則為 true  
  
 會報告這個警告時 _Satisfies\_所列的運算式不是 true。 通常這表示不正確的參數。  
  
 如果發生這種情況在函式宣告，附註會指出不可能條件。
