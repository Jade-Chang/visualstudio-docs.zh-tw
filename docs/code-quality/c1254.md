# <a name="c1254"></a>C1254
警告 C1254： 外掛程式版本不相符： 版本不符合與 PREfast 驅動程式版本  
  
 外掛程式版本，不在所分析的程式碼中發生內部錯誤時，程式碼分析工具會報告這個警告。
