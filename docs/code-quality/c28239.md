# <a name="c28239"></a>C28239
警告 C28239： 參數上的註釋具有兩個值，stringValue 或 longValue;以及 paramn = xxx。 請只使用 paramn=xxx  
  
 PREfast 模型檔案中發生內部錯誤。 在一般使用時，應該不會發生這個警告。
