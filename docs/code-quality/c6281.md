# <a name="c6281"></a>C6281
警告 6281： 作業順序不正確： 關係運算子具有的優先順序高於位元運算子  
  
 這個警告表示可能發生的錯誤中運算子的優先順序。 這可能會產生不正確的結果。 您應該檢查優先順序，並使用括號釐清目的。 關係運算子 (\<，>， \<=、 > =、 = =、 ！ =) 優先順序高於位元運算子 (（& c) &#124; ^)。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
#include <stdlib.h>  
#define FORMAT 1  
#define TYPE 2  
  
void f(int input)  
{  
  if (FORMAT & TYPE != input)  
  {  
    // code...  
  }  
}  
```  
  
 下列程式碼會使用括號，若要更正這個警告：  
  
```  
#include <stdlib.h>  
#define FORMAT 1  
#define TYPE 2  
  
void f(int input)  
{  
  if ((FORMAT & TYPE) != input)  
  {  
    // code...  
  }  
}  
```
