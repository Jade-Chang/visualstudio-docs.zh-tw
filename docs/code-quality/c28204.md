# <a name="c28204"></a>C28204
警告 C28204:\<函式 >： 這個多載，而在另一個的其中之一\<檔名 > (\<列 >) 會註明\<paramname >： 必須註解兩者皆非。

 註釋中發生錯誤時，會報告這個警告。
