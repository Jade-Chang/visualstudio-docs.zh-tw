# <a name="c6296"></a>C6296
警告 C6296: for 迴圈： 迴圈主體僅執行一次  
  
 這個警告表示預期的迴圈可能無法運作。 當索引是不帶正負號和向下計數從零的迴圈時，其主體為只執行一次。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
```  
  
 若要修正這個警告，請使用下列程式碼：  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
