# <a name="c6295"></a>C6295
警告 C6295: for 迴圈定義錯誤：\<變數 > 的值為 「 最小 」 至 「 上限 」 的範圍。 無限執行迴圈  
  
 這個警告表示預期的迴圈可能無法運作。 for 迴圈會用 >= 針對零 (0) 測試不帶正負號的值。 結果永遠為 true，因此在迴圈為無限。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
void f( )  
{  
  for (unsigned int i = 100; i >= 0; i--)   
  {  
    // code ...  
  }  
}  
```  
  
 若要修正這個警告，請使用下列程式碼：  
  
```  
void f( )  
{  
   for (unsigned int i = 100; i > 0; i--)  
   {  
      // code ...  
   }  
}  
```
