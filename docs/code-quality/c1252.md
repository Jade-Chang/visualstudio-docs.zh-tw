# <a name="c1252"></a>C1252
警告 C1252： 外掛程式之間的循環或遺漏相依性： 需要 GUID  
  
 外掛程式相依性，不在所分析的程式碼中發生內部錯誤時，程式碼分析工具會報告這個警告。
