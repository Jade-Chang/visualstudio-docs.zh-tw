# <a name="c28223"></a>C28223
警告 C28223： 註釋，找不到預期的語彙基元/識別參數

 這個警告表示為註解參數必須是識別項，且遇到一些其他符號。 這通常表示編碼錯誤的註解巨集。 在這個位置中的 C 或 c + + 關鍵字的使用會造成這個錯誤。
