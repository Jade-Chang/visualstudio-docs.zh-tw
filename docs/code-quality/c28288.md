# <a name="c28288"></a>C28288
警告 C28288： 函式的語法錯誤 （_a） 中\_（） 註釋 （無效的參數名稱）

 程式碼分析工具會報告這個警告的時機`SAL_at`(`__drv_at`) 用註解，且參數運算式無法在目前內容中解譯。 這可能包括使用的拼字錯誤的參數或成員名稱或 「 傳回 」 或"this"關鍵字拼錯。
