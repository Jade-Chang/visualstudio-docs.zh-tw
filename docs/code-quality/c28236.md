# <a name="c28236"></a>C28236
警告 C28236： 註釋不能在運算式中  
  
 這個警告表示嘗試使用註解在 expression 內容中的宣告的函式。 這不能進行註解; 使用標準巨集這個警告不會出現在一般使用。
