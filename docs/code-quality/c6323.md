# <a name="c6323"></a>C6323
警告 C6323 使用布林型別上的算術運算子  
  
 如果布林資料類型上使用算術運算子，就會發生這個警告。 使用不正確的運算子可能會產生不正確的結果。 它也會指出程式設計人員的目的不反映在程式碼。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
int test(bool a, bool b)  
{  
    int c = a + b;     //C6323  
    return c;  
}  
```  
  
 若要修正這個警告，請使用正確的資料類型與運算子。  
  
```  
int test(int a, int b)  
{  
    int c = a + b;       
    return c;  
}  
```
