# <a name="c28226"></a>C28226
警告 C28226： 註釋不能也是 PrimOp （在目前宣告中）

 這個警告表示具名註解被宣告為 PrimOp，和也先前宣告為一般的註解。 這不能進行註解; 使用標準巨集這個警告不會出現在正常使用。
