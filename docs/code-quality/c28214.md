# <a name="c28214"></a>C28214
警告 C28214： 屬性參數名稱必須是 p1 … 9 頁所述

 這則警告表示當您建構的註釋宣告，參數名稱會限制為 p1 … 9 頁所述。 在正常使用，應該永遠不會發生這個警告。
