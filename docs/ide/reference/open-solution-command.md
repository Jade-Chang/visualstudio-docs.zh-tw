# <a name="open-solution-command"></a>開啟方案命令
關閉任何其他開啟的方案，以開啟現有方案。  
  
## <a name="syntax"></a>語法  
  
```  
File.OpenSolution filename  
```  
  
## <a name="arguments"></a>引數  
 `Filename`  
 必要。 要開啟之方案的完整路徑和檔名。  
  
 `filename` 引數的語法需要包含空格的路徑使用引號。  
  
## <a name="remarks"></a>備註  
 自動完成會在您輸入時嘗試找出正確的路徑和檔名。  
  
## <a name="example"></a>範例  
 此範例會開啟 Test1.sln 方案。  
  
```  
>File.OpenSolution "c:\MySolutions\Test1\Test1.sln"  
```
