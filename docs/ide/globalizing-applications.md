# <a name="globalizing-applications"></a>全球化應用程式
「全球化」是針對多種文化特性，設計和開發軟體產品的程序。 下列文章同時適用於 Windows Form 和 Web Form 頁面。  
  
 [全域 Windows Form 和 Web Form 的文化特性 (Culture) 特定類別](../ide/culture-specific-classes-for-global-windows-forms-and-web-forms.md)  
 根據文化特性設定，探討可格式化日期、時間、數字、貨幣和其他資訊的類別。
